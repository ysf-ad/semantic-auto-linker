# Project Brief

Semantic Auto-Linker is meant to turn a sparse vault into a denser knowledge graph with safe reviewable edits.
The initial launch should highlight the difference between deterministic linking, graph preview, and later semantic reranking.
This brief references Market Positioning, Release Checklist, Review Workflow, and Graph Preview as separate notes.

The product should feel useful for note-writing workflows, personal knowledge management, and research vault maintenance.
